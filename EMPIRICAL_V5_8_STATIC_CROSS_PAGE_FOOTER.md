# Empirical v5.8 — Static Cross-Page Footer

- Footer markup is physically present as a direct child of `<body>` on every active page.
- No JavaScript is required to create, move, or reveal the footer.
- Critical footer styling is embedded in every page, preventing missing/cached shared assets from hiding it.
- Applied to Home, Archive, Categories, Location, About, Explore, and Editions 1–5.
