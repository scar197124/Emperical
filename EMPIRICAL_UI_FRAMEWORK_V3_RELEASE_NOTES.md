# Empirical UI Framework v3.0

## Stable workspace
- Fixed-height home reading card and complete-edition navigator.
- Story and genre selections update content without changing page geometry.
- Primary Source remains anchored at the bottom of the reading frame.
- Long story text and story lists scroll internally.
- Mobile uses fixed reading and navigator frames stacked vertically.

## Shared publication behavior
- Archive, Categories, and Location now use fixed-height cards with internal scrolling.
- Section headings are renamed Archived Editions, Browse by Category, and Browse by Location.
- Active navigation receives a consistent gold state.
- Existing links, datasets, archived editions, artwork, and the All 20 Stories image are preserved.
